Project info

Design and architecture

- Layout and design: HTML and CSS
- Java Script library: jQuery
- Visualisation library: VegaLite
- Data storage: Excel and CSV
- Data wrangling and anlysiss: Excel and R

Inputs and outputs

- We transformed the data provided by St Kilda Football Club
- Made specific CSV files for different analyses
- Input transformed data to Excel and R, output labelled data
- Input labelled data to VegaLite, output visualisations
- Input visualisations and code into website, output user interface

Major requirements

- Provide insight by labelling decisions as good or bad
- Provide analysis by explaining factors leading to certain decisions
- Present information in an easy to understand format for client

Limitations

- Product does not have the ability to receive new data
- Inherent football complexity cannot be fully explained through data provided by client
- Analysis are not "live", but rather are static and presented to the client

Our partner：St Kilda Football Club

The St Kilda Football Club, nicknamed the Saints, is a professional Australian rules football club based in Melbourne, Victoria. The club plays in the Australian Football League (AFL), the sport's premier league. The club's name originates from its original home base in the bayside Melbourne suburb of St Kilda in which the club was established in 1873. The club also has strong links to the south-eastern suburb of Moorabbin, due to it being the long-standing location of their training ground.
